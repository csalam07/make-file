# make-file

An alternative for touch cmd (Linux) for Windows OS.

## Installation

    $ npm install -g make-file

## Usage

```
make [filename]
```

## Examples

Create a zero-sized new file:

    $ make my_new_file

Update the last modified and access timestamps of an existing file:

    $ make my_existing_file
